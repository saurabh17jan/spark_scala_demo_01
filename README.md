# spark_scala_demo_01
Find Word and its frequency

## Objective
The idea is to give a demo for following features

* Perform Spark Dataframe Transformation
* Perform Word Count for a CSV file to demo RDD feature
* Parse CLI args
* Demo logging feature
* Demo UDF feature

 scalaVersion: 2.11.8
