# spark_scala_demo_01
Find Word and its frequency

## Objective
The idea is to give a demo for following features

* Perform simple Spark Dataframe Transformation
* Perform Word Count for a CSV file to demo RDD feature
* Read CLI args
* Set logging setup
* Demo UDF feature
* Case Class

## Note
```bash
 scalaVersion: 2.11.8
 spark: 2.0.1
 build: sbt
```
